# 🚀 PIXEL OFFICE — MASTER BLUEPRINT
*Last Updated: April 3, 2026 (22:12 UTC)*

Pixel Office is a browser-based, AI-native **Company Operating System** with pixel-art aesthetics. Each agent runs on an **AI Brain** (LLM).

---

## 🏗️ ARCHITECTURE
- **Renderer:** HTML5 Canvas (60 FPS, Pixel Art)
- **Style:** Retro, Dark Theme, Monospace (No rounded corners, No gradients)
- **Data:** localStorage persistence (17 keys)
- **AI:** OpenAI-compatible `/chat/completions` (6+ providers)
- **GitHub:** 3-tier repo system (Standalone/Ref/Latest)

---

## 👥 HR SYSTEM
- **12 Agent Roles:** CEO, CTO, Manager, Developer, DevOps, Designer, QA, HR, Marketing, Sales, Analyst, Intern
- **Hire/Fire:** With country, currency, salary, AI config
- **Status:** Working → Meeting → Break → Idle
- **Promote/Demote:** Level-based hierarchy

---

## 🤖 AI BRAIN SYSTEM
- **6 Providers:** LiteLLM, OpenAI, Groq, Ollama, OpenRouter, Custom
- **Per-Agent Config:** Provider, URL, API Key, Model
- **Context-Aware:** Memory + Skills injected in prompts
- **Agent Chat:** Per-agent conversations with history

---

## 📊 COMPANY DASHBOARD (8 Tabs)
1. **📋 Task Board** — Kanban, AI execution, Autopilot, Goal setting
2. **📁 Output Folders** — Per-agent task outputs, preview, download
3. **🧑‍💼 Scrum** — AI daily standups, CEO summary, history
4. **📈 Analytics** — KPIs, performance bars, dept distribution
5. **👥 Team** — Agent cards, LVL/XP badges, memory indicator
6. **🗂️ Org Chart** — Visual hierarchy tree
7. **🏢 Clients** — Client management, projects, revenue
8. **👑 CEO Inbox** — Messages, file routing, AI responses

---

## 🧠 AGENT MEMORY SYSTEM
- **Memory Store:** Last 20 completed tasks per agent
- **Skill Growth:** XP system (+1 per task) across 7 skill types
- **21 Badges:** 3 tiers per skill (Jr Dev → Sr Dev → Tech Lead, etc.)
- **Memory Viewer:** Full skills + history UI
- **CEO Brief:** Smart task assignment using team skills

---

## 🔄 AUTOPILOT MODE
- CEO sets Company Goal (title + description)
- "🤖 Autopilot Tasks" → CEO AI generates tasks
- "🟢 Auto-Run ON" → 30s loop auto-executes tasks
- Task complete → Agent memory updated + CEO notified
- "⏸️ Pause/Resume" toggle

---

## 🏢 CLIENT/CUSTOMER SYSTEM
- **Client Portal:** Add clients with budget, projects
- **Project Lifecycle:** Briefing → In Progress → Review → Delivered
- **Revenue:** Auto-add to company balance on delivery
- **Satisfaction:** 1-5 star ratings
- **Analytics:** Total revenue, avg rating, recent deliveries

---

## 💬 AGENT GROUP CHAT
- **Team Chat Panel:** Right-side panel, CEO/Announcement modes
- **@Mention System:** Dropdown suggestions, yellow highlights
- **AI Responses:** Agents auto-respond when mentioned
- **Typing Indicator:** "Agent is typing..." animation
- **Unread Badge:** Message counter
